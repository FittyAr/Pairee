# Support Guidelines

Thank you for using Pairee! If you need help with Pairee or its plugins, please check the resources below.

## 📖 Documentation

Before opening a support request, we recommend reading the official documentation:
- [User Guide (English)](https://github.com/FittyAr/Pairee/blob/master/help/en/user_guide.md)
- [Guía del Usuario (Spanish)](https://github.com/FittyAr/Pairee/blob/master/help/es/user_guide.md)
- [Plugin Developer Guide](https://github.com/FittyAr/Pairee/blob/master/docs/plugin-dev-guide.md)

## 💬 Community Support

If you have questions, need troubleshooting help, or want to discuss configuration ideas:
- **GitHub Discussions**: Visit the [Pairee Discussions page](https://github.com/FittyAr/Pairee/discussions) to ask questions and engage with the community.
- **Matrix / Chat**: Join our community chat room (if configured).

## 🪲 Reporting Bugs & Issues

If you've found a bug or want to request a feature:
- Check the [Issue Tracker](https://github.com/FittyAr/Pairee/issues) to see if it has already been reported.
- If not, open a new issue with a clear description, reproduction steps, and configuration details.
